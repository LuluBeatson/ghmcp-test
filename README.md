# GHMCP Test Repository

A test repository for GitHub Model Context Protocol (MCP) functionality and testing various file operations.

## Overview

This repository contains sample files of various sizes and formats used for testing GitHub MCP server capabilities, file handling, and integration workflows.

## Contents

- **Image Assets**: Multiple image files in different formats and sizes
  - `7kb.jpg` - Small JPEG image (7KB)
  - `300kb.jpg` - Medium JPEG image (300KB) 
  - `10mb.png` - Large PNG image (10MB)
  - `20mb.jpg` - Very large JPEG image (20MB)
